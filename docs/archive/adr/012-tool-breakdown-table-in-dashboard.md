# ADR-012: ダッシュボードにツール別 permission UI テーブルを追加

## ステータス

採用済み

## 関連 ADR

- 依存: ADR-009（ツール別 permission UI 内訳の棒グラフを追加）

## コンテキスト

ADR-009 でダッシュボードにツール別 permission UI 内訳の棒グラフが追加された。しかし棒グラフだけでは件数・割合の正確な値が読み取りにくく、「次に何を allowlist に追加すべきか」の判断材料として不十分である。

棒グラフに加えてテーブル（件数 + 割合）を併記すれば、ツールごとの寄与度が一目で分かり改善施策の優先順位付けが容易になる。

## 設計案

既存の `aggregate_by_tool` の結果を棒グラフの下にテーブルとして出力する。

- カラム: ツール名 / 件数 / 割合（%）
- ソート: 件数降順
- 棒グラフは既存のまま維持し、テーブルを直下に追加する

### 変更が必要なファイル

| ファイル | リポジトリ | 変更内容 |
|---|---|---|
| `configs/claude/scripts/permission-ui-server.py` | dotfiles | `generate_tool_table` 関数を追加し、ツール別セクションにテーブルを併記 |

## 受け入れ条件

- [x] ツール別テーブルがダッシュボードに表示される
